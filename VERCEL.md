# Publicar a Agenda Josielly no Vercel

## Opção mais simples

1. Acesse `https://vercel.com`.
2. Crie uma conta ou entre com sua conta.
3. Clique em **Add New... > Project**.
4. Envie/conecte a pasta `AGENDA`.
5. Framework: **Other**.
6. Build Command: deixar vazio.
7. Output Directory: deixar vazio.
8. Clique em **Deploy**.

Depois do deploy, o Vercel vai gerar um link parecido com:

```text
https://agenda-josielly.vercel.app
```

## Instalar no iPhone

1. Abra o link da agenda no Safari do iPhone.
2. Toque no botão de compartilhar.
3. Toque em **Adicionar à Tela de Início**.
4. Confirme.

Assim a agenda aparece como aplicativo no iPhone.

## Importante

Publicar no Vercel deixa a agenda acessível em qualquer lugar, mas os dados ainda estarão salvos no navegador enquanto não conectarmos um banco em nuvem.

Para sincronizar agendamentos entre computador e iPhone, a próxima etapa é conectar um banco, como Supabase.
